# Support

## Questions and usage help

Use GitHub Issues for:

- bug reports
- documentation gaps
- source adapter problems
- local setup trouble
- improvement suggestions

## Security issues

Use the private path in [SECURITY.md](SECURITY.md) instead of opening a public
issue.

## Scope of support

This repository supports the public codebase, local development workflow, and
reproducible deployment contracts. Private infrastructure inventory, live host
operations, and server-specific credentials are intentionally outside scope.
