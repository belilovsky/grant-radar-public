# Historical release evidence

This directory contains dated launch plans and production closeout snapshots.
They describe the system as it existed on the date in each filename and may
mention services or counts that have since changed.

Use the following documents for current operations:

- [`DEPLOYMENT.md`](../../../DEPLOYMENT.md)
- [`docs/PRODUCTION_CHECKLIST.md`](../../PRODUCTION_CHECKLIST.md)
- [`docs/DEVELOPMENT_MODEL.md`](../../DEVELOPMENT_MODEL.md)

Historical files are retained for traceability. They are not active runbooks or
current backlog documents.
